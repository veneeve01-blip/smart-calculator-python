# Smart Calculator (Python)

## 📌 Description
A simple calculator built using Python that performs basic arithmetic operations.

## ⚙️ Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles divide by zero error

## 🛠️ Tech Used
- Python

## ▶️ How to Run
1. Download the file
2. Run main.py using Python
3. Follow the instructions on screen
